# 招聘运营数据下钻体系功能验收文档

## 1. 验收目标

本验收文档用于验证“招聘运营数据下钻体系”是否达到长期可维护、真实业务可扩展的建设要求。

系统需支持从招聘运营总览指标逐级下钻至平台、账号、内容、岗位和漏斗，确保指标口径统一、数据来源可追溯、权限控制有效、异常场景可处理，并能支撑后续接入小红书、脉脉、B站、公众号、抖音、知乎、技术社区、北森等真实业务数据。

## 2. 验收范围

本次验收覆盖以下模块：

- 数据分析总览
- 平台下钻
- 账号下钻
- 内容下钻
- 岗位下钻
- 漏斗下钻
- 指标口径计算
- 数据来源追踪
- 数据质量问题
- 后端 Analytics API
- 权限控制
- 缓存与分页
- 导出能力
- 异常处理

## 3. 验收环境

| 项目 | 要求 |
|---|---|
| 前端环境 | 招聘运营助手 Web 系统 |
| 后端环境 | 本地或内网 API 服务 |
| 数据环境 | 真实业务数据或验收测试数据 |
| 浏览器 | Chrome / Edge 最新稳定版 |
| 权限账号 | 管理员、招聘运营、招聘专员、招聘负责人、用人部门 |
| 数据来源 | 平台指标、北森回流、账号配置、岗位数据、内容数据、成本数据 |

## 4. 验收通过标准

系统满足以下条件即视为整体通过：

- 无真实数据时，所有指标显示为 0，不展示演示数据。
- 平台、账号、内容、岗位、漏斗均支持下钻。
- 所有页面使用统一指标口径。
- 点击指标卡、平台卡、账号行、内容行、岗位行可进入对应下钻。
- 数据来源、导入批次、同步状态可追踪。
- 数据质量问题可识别、展示和处理。
- API 返回结构稳定，支持筛选、分页和权限过滤。
- 不同角色只能查看授权范围内的数据。
- 导出结果与当前筛选条件一致。
- 核心计算、接口和交互测试全部通过。

## 6. 指标口径验收

| 指标 | 验收公式 | 验收标准 |
|---|---|---|
| 曝光 | views 求和 | 按 contentId + metricDate + sourceId 去重 |
| 互动 | likes + comments + saves + shares | 与平台明细求和一致 |
| 点击 | clicks 求和 | 与招聘入口点击一致 |
| 投递 | stage = 已投递 | candidateCode + jobId 去重 |
| 有效简历 | 有效简历及之后阶段 | 不重复计算同一候选人 |
| 面试 | 已约面 + 已面试 | 阶段映射正确 |
| Offer | Offer + 已入职 | 已入职应计入 Offer |
| 入职 | stage = 已入职 | 去重后人数正确 |
| 互动率 | 互动 / 曝光 | 分母为 0 时显示 0% |
| 点击率 | 点击 / 曝光 | 分母为 0 时显示 0% |
| 投递转化率 | 投递 / 点击 | 分母为 0 时显示 0% |
| 有效简历率 | 有效简历 / 投递 | 分母为 0 时显示 0% |
| 入职转化率 | 入职 / 投递 | 分母为 0 时显示 0% |
| 总成本 | 人工 + 投放 + 制作 | 按 targetType + targetId + costDate 汇总 |
| 单入职成本 | 总成本 / 入职 | 入职为 0 时显示 0 |

## 7. 页面交互验收用例

### 用例 1：无数据总览

前置条件：

- 系统无内容数据。
- 系统无平台指标。
- 系统无北森回流。

操作步骤：

1. 登录系统。
2. 进入“数据分析”页面。

预期结果：

- 曝光、互动、点击、投递、有效简历、入职均为 0。
- 页面无报错。
- 页面提示导入平台指标、导入北森回流或配置 API。

### 用例 2：平台下钻

前置条件：

- 小红书存在 3 条内容指标。
- 内容已关联岗位和账号。

操作步骤：

1. 进入“数据分析”。
2. 点击“小红书”平台卡片。

预期结果：

- 打开小红书平台下钻。
- 展示小红书内容数、账号数、曝光、互动、点击、投递、入职。
- 展示该平台内容明细。
- 可继续点击内容或账号进入下一级下钻。

### 用例 3：账号下钻

前置条件：

- 某小红书账号已配置。
- 该账号关联多条已发布内容。

操作步骤：

1. 进入平台下钻。
2. 点击账号明细。

预期结果：

- 展示账号名称、定位、负责人、授权状态。
- 展示发布内容数、最近发布时间、停更天数。
- 展示平均曝光、互动率、点击率。
- 展示账号健康度和改进建议。

### 用例 4：内容下钻

前置条件：

- 某内容存在曝光、互动、点击数据。
- 北森回流中存在该内容带来的投递。

操作步骤：

1. 进入内容明细列表。
2. 点击某条内容。

预期结果：

- 展示内容标题、平台、账号、岗位、内容类型。
- 展示曝光、互动、点击、点击率、互动率。
- 展示北森回流阶段分布。
- 展示风险等级、版本记录和复盘建议。

### 用例 5：岗位下钻

前置条件：

- 某岗位关联多个平台内容。
- 北森存在该岗位候选人回流。

操作步骤：

1. 进入“岗位下钻”。
2. 选择目标岗位。

预期结果：

- 展示岗位基础信息。
- 展示各平台内容覆盖情况。
- 展示平台贡献排行。
- 展示内容类型贡献排行。
- 展示候选人阶段分布。

### 用例 6：漏斗断点识别

前置条件：

- 某平台曝光较高。
- 点击量为 0 或明显偏低。

操作步骤：

1. 进入漏斗下钻。
2. 查看曝光到点击环节。

预期结果：

- 系统标记曝光到点击为异常环节。
- 给出“CTA、招聘入口、内容落点可能存在问题”的提示。
- 可点击查看相关平台、内容和账号。

### 用例 7：数据质量问题识别

前置条件：

- 导入一批缺少 contentId 或 jobId 的数据。

操作步骤：

1. 在导入中心导入数据。
2. 进入数据质量问题列表。

预期结果：

- 系统识别缺字段问题。
- 问题包含来源、批次、目标类型、错误说明。
- 无法归因的数据不进入错误指标统计。

### 用例 8：重复导入

前置条件：

- 已存在 contentId + metricDate + sourceId 相同的指标数据。

操作步骤：

1. 再次导入同一批数据。

预期结果：

- 系统按唯一键去重。
- 指标不重复累加。
- 同步批次记录重复数量。

### 用例 9：权限过滤

前置条件：

- 招聘专员 A 只负责岗位 A。
- 系统中同时存在岗位 A 和岗位 B 的数据。

操作步骤：

1. 使用招聘专员 A 登录。
2. 进入数据分析。

预期结果：

- 只能看到岗位 A 相关数据。
- 无法查看岗位 B 明细。
- 访问无权限 API 返回 403。

### 用例 10：导出当前筛选结果

前置条件：

- 当前筛选条件为“小红书 + 2026-05-01 至 2026-05-31”。

操作步骤：

1. 点击导出。
2. 下载导出文件。

预期结果：

- 导出文件只包含当前筛选范围内的数据。
- 指标与页面展示一致。
- 导出任务状态可查询。

## 8. API 验收用例

| 接口 | 验收内容 | 预期结果 |
|---|---|---|
| GET /api/analytics/summary | 查询总览 | 返回 summary、insights、qualityIssues |
| POST /api/analytics/drill/platform | 平台下钻 | 返回平台 summary、breakdowns、details |
| POST /api/analytics/drill/account | 账号下钻 | 返回账号健康度和内容明细 |
| POST /api/analytics/drill/content | 内容下钻 | 返回单内容指标和北森阶段 |
| POST /api/analytics/drill/job | 岗位下钻 | 返回岗位平台贡献和内容覆盖 |
| POST /api/analytics/drill/funnel | 漏斗下钻 | 返回完整转化漏斗 |
| GET /api/analytics/quality-issues | 质量问题 | 返回未处理质量问题 |
| POST /api/analytics/export | 导出 | 返回导出任务 ID 和下载地址 |

API 通用验收标准：

- 请求参数校验完整。
- 无权限返回 403。
- 参数错误返回 400。
- 服务异常返回 500。
- 返回结构符合 `DrillResult`。
- 分页字段正确。
- 无数据时返回空数组和 0 指标。

## 9. 权限验收

| 角色 | 验收标准 |
|---|---|
| 招聘专员 | 只能查看本人负责岗位、内容、账号 |
| 招聘运营 | 可查看全部运营数据，可处理数据质量问题 |
| 招聘负责人 | 可查看全部汇总、ROI 和趋势 |
| 用人部门 | 只能查看关联岗位和关联内容 |
| 管理员 | 可查看和管理全部数据、接口、权限、导出 |

## 10. 异常验收

| 异常场景 | 预期处理 |
|---|---|
| API 请求失败 | 页面提示失败，允许重试 |
| 部分数据同步失败 | 成功数据入库，失败数据进入质量问题 |
| 字段缺失 | 阻断或标记质量问题 |
| 重复数据 | 去重，不重复累加 |
| 无法归因 | 进入质量问题，不污染指标 |
| 日期异常 | 标记为中风险问题 |
| 分母为 0 | 转化率显示 0% |
| 权限不足 | 返回 403，页面展示无权限 |
| 缓存失效 | 自动重新计算 |
| 导出超时 | 转为异步任务 |

## 11. 性能验收

| 场景 | 验收标准 |
|---|---|
| 总览查询 | 3 秒内返回 |
| 下钻查询 | 5 秒内返回 |
| 明细分页 | 每页默认 20 条，可切换分页 |
| 导出大文件 | 支持异步任务 |
| 缓存命中 | 返回时间明显低于首次查询 |
| 导入后刷新 | 相关缓存自动失效 |
