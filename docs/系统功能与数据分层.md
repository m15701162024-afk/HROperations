# 系统功能与数据分层

本系统面向招聘运营专员，一级入口只保留招聘运营的真实工作场景：运营首页、内容工厂、渠道数据、连接配置。其余页面不再作为并列入口出现，而是按“业务入口 -> 工作场景 -> 操作能力 -> 数据指标”分层收纳；同一一级入口只出现一次，入口下再展开多个工作场景。

## 分层总览

| 第一层业务入口 | 第二层工作场景 | 第三层操作能力 | 第四层数据指标 |
| --- | --- | --- | --- |
| 运营首页 | 今日指挥台 | 今日任务、异常提醒、推荐动作、快捷跳转 | 待办数量、待发布内容、待审核内容、数据待回收内容、招聘入口点击、运营目标进度 |
| 内容工厂 | 岗位经营 | 岗位录入、岗位编辑、批量导入、状态管理、北森/官网入口维护 | 招聘中岗位、暂停/关闭岗位、目标平台覆盖、岗位入口完整度、岗位关联内容数、北森回流数 |
| 内容工厂 | 选题策划 | 岗位生成选题、渠道选题储备、选题转内容、平台选题管理 | 选题数量、关联岗位数、目标平台分布、选题转内容数、待生产选题 |
| 内容工厂 | 内容生产 | 多平台内容生成、风险扫描、内容任务创建、版本修订、审核流转、发布前检查 | 内容任务数、状态分布、风险等级、质量评分、审核意见数、待发布内容、已发布内容 |
| 内容工厂 | 发布排期 | 内容排期查看、平台/账号筛选、排期日期调整、发布冲突检查 | 排期内容数、账号日发布量、平台周频次、排期冲突数、发布延迟数 |
| 内容工厂 | 素材管理 | 素材库、授权记录、素材上传、采集表、模板案例库 | 素材数量、授权状态、高风险素材、素材使用次数、到期素材、文件上传记录 |
| 内容工厂 | 智能辅助 | 提示词模板、模型任务执行、模型运行记录、生成内容辅助 | 模型任务数、提示词模板数、成功运行数、失败运行数、平均耗时、生成内容采纳数 |
| 渠道数据 | 数据接入 | CSV 导入、字段映射、数据预检、重复识别、未归因识别 | 导入批次数、成功记录数、失败记录数、字段映射完整度、重复数据、未归因指标 |
| 渠道数据 | 渠道表现 | 平台效果分析、账号效果分析、内容/岗位下钻、ROI 分析 | 曝光、互动、点击、ROI、数据质量问题 |
| 渠道数据 | 转化归因 | 漏斗归因、北森结果关联、岗位归因、内容归因 | 投递、有效简历、面试、Offer、入职 |
| 渠道数据 | 线索转化 | 线索录入、线索导入、重复识别、负责人分配、批量转北森 | 总线索、待联系线索、重复线索、已转北森线索、岗位归因线索、线索负责人分布 |
| 渠道数据 | 复盘决策 | 周报/月报生成、机会风险建议、行动项管理、高表现内容特征提炼 | 报告数量、机会/风险/建议数、行动项完成率、高表现内容特征、周报导出次数 |
| 连接配置 | 平台连接 | 平台账号管理、授权状态维护、账号健康检查、同步日志查看 | 平台账号数、授权状态、账号健康分、同步成功率 |
| 连接配置 | 招聘入口 | 招聘入口配置、落地页管理、访问点击统计、留资回收 | 招聘入口点击、落地页访问、落地页点击、落地页留资 |
| 连接配置 | 外部集成 | 北森集成、企业微信集成、飞书集成、平台 API 配置、大模型 API 测试 | 集成数量、连接状态、同步成功率、失败原因 |
| 连接配置 | 系统治理 | 角色权限、用户管理、审核流程、敏感词规则、运营参数、系统健康和备份 | 角色数量、用户数量、流程规则数、敏感词规则数、上线任务进度、系统备份数 |

## 数据归类

| 数据类别 | 数据表/对象 | 使用场景 |
| --- | --- | --- |
| 招聘需求数据 | `jobs` | 岗位经营、内容生产、选题策划、渠道表现、线索转化 |
| 内容生产数据 | `contents`、`contentVersions`、`reviewComments`、`contentQualityScores`、`reviewMentions` | 内容生产、发布排期、渠道表现、转化归因、复盘决策 |
| 渠道账号数据 | `accounts`、`accountHealthSnapshots`、`entries`、`landingPages`、`landingLeads` | 平台连接、招聘入口、内容生产、渠道表现、转化归因 |
| 素材与选题数据 | `assets`、`topics`、`promptTemplates`、`modelRunLogs` | 素材管理、选题策划、智能辅助、内容生产 |
| 指标与归因数据 | `beisenResults`、`costs`、`dataExplanations`、`importRuns`、`integrationMappings` | 数据接入、渠道表现、转化归因、复盘决策 |
| 线索跟进数据 | `candidateLeads`、`leadFollowUps` | 线索转化、转化归因、复盘决策 |
| 复盘行动数据 | `reports`、`reportActions`、`goals`、`tasks`、`taskCompletions`、`notifications` | 今日指挥台、复盘决策 |
| 连接与系统数据 | `integrations`、`integrationSyncRuns`、`modelApis`、`roles`、`users`、`workflowRules`、`sensitiveRules`、`compliancePolicies`、`deploymentTasks`、`pluginRules`、`auditLogs`、`operationSettings` | 平台连接、外部集成、系统治理、今日指挥台 |

## 跳转归属

所有旧页面仍保留直接路由能力，但侧边栏只高亮其所属一级入口：

| 直接目标 | 所属入口 | 默认工作场景 |
| --- | --- | --- |
| 工作台 | 运营首页 | 今日指挥台 |
| 招聘需求 | 内容工厂 | 岗位经营 |
| 选题库 | 内容工厂 | 选题策划 |
| 内容运营 | 内容工厂 | 内容生产 |
| 排期日历 | 内容工厂 | 发布排期 |
| 素材资产 | 内容工厂 | 素材管理 |
| AI工作台 | 内容工厂 | 智能辅助 |
| 导入中心 | 渠道数据 | 数据接入 |
| 数据分析 | 渠道数据 | 渠道表现 / 转化归因 |
| 线索池 | 渠道数据 | 线索转化 |
| 复盘报告 | 渠道数据 | 复盘决策 |
| 账号与平台 | 连接配置 | 平台连接 / 招聘入口 / 外部集成 |
| 系统配置 | 连接配置 | 系统治理 |

首页待办、通知和快捷按钮可以直接打开具体页签；进入后仍停留在其所属一级入口下，避免侧边栏出现重复入口。

## 权限判断

一级入口按其下工作场景的权限聚合判断：用户只要拥有该入口下任一场景权限，就能看到一级入口。进入模块后，工作场景再按用户权限过滤，只展示可操作场景；场景中的操作能力和数据指标随当前场景展示。

| 工作场景 | 权限 |
| --- | --- |
| 今日指挥台 | 工作台查看 |
| 岗位经营 | 岗位查看 |
| 选题策划 | 内容创建 |
| 内容生产、发布排期 | 内容查看 |
| 素材管理 | 素材查看 |
| 智能辅助 | AI配置 |
| 数据接入 | 数据导入 |
| 渠道表现、转化归因 | 数据查看 |
| 线索转化 | 岗位查看 |
| 复盘决策 | 复盘查看 |
| 平台连接、招聘入口、外部集成 | 账号查看 |
| 系统治理 | 系统配置 |
