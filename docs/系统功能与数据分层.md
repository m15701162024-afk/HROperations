# 系统功能与数据分层

本系统面向招聘运营专员，一级入口只保留招聘运营的真实工作场景：运营首页、内容工厂、渠道数据、连接配置。其余页面不再作为并列入口出现，而是按工作类型收纳为二级页签；每个页签下方展示该能力依赖和产出的数据指标。

## 分层总览

| 一级入口 | 二级页签 | 数据指标 |
| --- | --- | --- |
| 运营首页 | 今日待办 | 待办数量、待发布内容、待审核内容、数据待回收内容、招聘入口点击、运营目标进度 |
| 内容工厂 | 内容生产 | 内容任务数、状态分布、风险等级、质量评分、审核意见数、待发布内容、已发布内容 |
| 内容工厂 | 岗位需求 | 招聘中岗位、暂停/关闭岗位、目标平台覆盖、岗位入口完整度、岗位关联内容数、北森回流数 |
| 内容工厂 | 选题库 | 选题数量、关联岗位数、目标平台分布、选题转内容数、待生产选题 |
| 内容工厂 | 排期日历 | 排期内容数、账号日发布量、平台周频次、排期冲突数、发布延迟数 |
| 内容工厂 | 素材资产 | 素材数量、授权状态、高风险素材、素材使用次数、到期素材、文件上传记录 |
| 内容工厂 | AI工作台 | 模型任务数、提示词模板数、成功运行数、失败运行数、平均耗时、生成内容采纳数 |
| 渠道数据 | 渠道分析 | 曝光、互动、点击、投递、有效简历、面试、Offer、入职、ROI、数据质量问题 |
| 渠道数据 | 导入中心 | 导入批次数、成功记录数、失败记录数、字段映射完整度、重复数据、未归因指标 |
| 渠道数据 | 线索池 | 总线索、待联系线索、重复线索、已转北森线索、岗位归因线索、线索负责人分布 |
| 渠道数据 | 复盘报告 | 报告数量、机会/风险/建议数、行动项完成率、高表现内容特征、周报导出次数 |
| 连接配置 | 账号与API | 平台账号数、授权状态、账号健康分、同步成功率、招聘入口点击、落地页访问/点击/留资 |
| 连接配置 | 系统配置 | 角色数量、用户数量、流程规则数、敏感词规则数、上线任务进度、系统备份数 |

## 数据归类

| 数据类别 | 数据表/对象 | 使用位置 |
| --- | --- | --- |
| 招聘需求数据 | `jobs` | 岗位需求、内容生产、选题库、渠道分析、线索池 |
| 内容生产数据 | `contents`、`contentVersions`、`reviewComments`、`contentQualityScores`、`reviewMentions` | 内容生产、排期日历、渠道分析、复盘报告 |
| 渠道账号数据 | `accounts`、`accountHealthSnapshots`、`entries`、`landingPages`、`landingLeads` | 账号与API、内容生产、渠道分析 |
| 素材与选题数据 | `assets`、`topics`、`promptTemplates`、`modelRunLogs` | 素材资产、选题库、AI工作台、内容生产 |
| 指标与归因数据 | `beisenResults`、`costs`、`dataExplanations`、`importRuns`、`integrationMappings` | 导入中心、渠道分析、复盘报告 |
| 线索跟进数据 | `candidateLeads`、`leadFollowUps` | 线索池、渠道分析、复盘报告 |
| 复盘行动数据 | `reports`、`reportActions`、`goals`、`tasks`、`taskCompletions`、`notifications` | 运营首页、复盘报告 |
| 连接与系统数据 | `integrations`、`integrationSyncRuns`、`modelApis`、`roles`、`users`、`workflowRules`、`sensitiveRules`、`compliancePolicies`、`deploymentTasks`、`pluginRules`、`auditLogs`、`operationSettings` | 连接配置、系统配置、运营首页 |

## 跳转归属

所有旧页面仍保留直接路由能力，但侧边栏只高亮其所属一级入口：

| 直接目标 | 所属入口 |
| --- | --- |
| 工作台 | 运营首页 |
| 招聘需求、选题库、内容运营、排期日历、素材资产、AI工作台 | 内容工厂 |
| 导入中心、线索池、数据分析、复盘报告 | 渠道数据 |
| 账号与平台、系统配置 | 连接配置 |

首页待办、通知和快捷按钮可以直接打开具体页签；进入后仍停留在其所属一级入口下，避免侧边栏出现重复入口。

## 权限判断

一级入口按其下二级页签的权限聚合判断：用户只要拥有该入口下任一页签权限，就能看到一级入口。进入模块后，二级页签再按用户权限过滤，只展示可操作页签。

| 页签 | 权限 |
| --- | --- |
| 运营首页 | 工作台查看 |
| 岗位需求 | 岗位查看 |
| 选题库 | 内容创建 |
| 内容生产、排期日历 | 内容查看 |
| 素材资产 | 素材查看 |
| AI工作台 | AI配置 |
| 导入中心 | 数据导入 |
| 渠道分析 | 数据查看 |
| 线索池 | 岗位查看 |
| 复盘报告 | 复盘查看 |
| 账号与API | 账号查看 |
| 系统配置 | 系统配置 |
